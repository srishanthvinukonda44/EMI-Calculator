# 💰 EMI Calculator

## 📌 Overview
The **EMI Calculator** is a simple web application that helps users calculate their monthly loan payments (Equated Monthly Installments).  
By entering the loan amount, interest rate, and tenure, the calculator instantly shows the EMI value with a clean and responsive interface.

## 🚀 Features
- Input fields for loan amount, interest rate, and tenure
- Instant EMI calculation using JavaScript
- Responsive design with HTML & CSS
- Easy to use for students, professionals, and developers

## 🛠️ Tech Stack
- **HTML** – Structure of the webpage  
- **CSS** – Styling and layout  
- **JavaScript** – EMI calculation logic
