🏦 Loan EMI Calculator

A simple and responsive Loan EMI Calculator built using HTML, CSS, and JavaScript. This application helps users calculate their monthly EMI (Equated Monthly Installment) based on the loan amount, interest rate, and loan tenure.

📸 Project Preview

The application provides:

- Loan Amount Input (₹)
- Interest Rate Input (% per year)
- Loan Tenure Input (Years)
- Instant EMI Calculation
- Clean and User-Friendly Interface

✨ Features

- Easy-to-use design
- Responsive layout
- Real-time EMI calculation
- Accurate loan repayment estimation
- Mobile-friendly interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

🚀 How to Run

1. Clone the repository:
   
   git clone https://github.com/srishanthvinukonda44/EMI-Calculator
2. Open the project folder.

3. Open "index.html" in your web browser.

OR

4. Run with VS Code Live Server.

📖 How It Works

1. Enter the loan amount.
2. Enter the annual interest rate.
3. Enter the loan tenure in years.
4. Click the Calculate EMI button.
5. View the calculated monthly EMI amount.
